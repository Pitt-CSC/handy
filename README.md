# Handy

Handy takes attendance at events via SMS.

[![Travis CI][1]][2]


## Getting Started

Install the system services Handy relies on:

* MySQL 5.6

Clone the Handy repository and run `bin/setup`:

```bash
git clone git@github.com:Pitt-CSC/handy.git
cd handy
bin/setup
```

That's it!
